REPETITION

This is a paragraph.

This is another paragraph.

This is yet another paragraph.

But see how none of them have the same font? Using a repeated font for things that are the same, like paragraphs, can be helpful. Above, font was used to show contrast, but in this case, these paragraphs are intended to come together to make a story. Using the same font makes a cohesive look that helps users follow your story.
